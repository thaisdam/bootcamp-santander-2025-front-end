Conhecendo as Principais Tags HTML

O HTML, ou HyperText Markup Language, é a linguagem de marcação que estrutura o conteúdo na web. Ele é composto por tags que indicam a função de cada elemento em uma página.

Neste desafio, vamos explorar algumas das tags mais comuns e entender como elas são aplicadas.

Tags de Títulos

As tags de título, de <h1> a <h6>, são usadas para definir a hierarquia de títulos e subtítulos em uma página. O <h1> é o título mais importante, e o <h6> o menos importante.

Tags de Formatação de Texto

Essas tags permitem dar ênfase e estilizar o texto de diferentes maneiras.

  • A tag <p> define um parágrafo.
  • A tag <mark> destaca um trecho de texto.
  • A tag <small> diminui o tamanho da fonte, ideal para notas de rodapé.
  • A tag <i> deixa o texto em itálico.
  • A tag <u> sublinha o texto (note que <u> é geralmente usado para links, mas pode ser usado para sublinhar).
  • A tag <strong> indica que um texto é importante ou urgente.
  • A tag <del> mostra que um texto foi deletado ou está incorreto.
  • A tag <sub> cria um subscrito, como em H2O.
  • A tag <sup> cria um sobrescrito, como em 210.

Tags de Listas

Existem dois tipos principais de listas: ordenadas e não ordenadas.

Lista Ordenada (<ol> e <li>)

Usada para itens em uma sequência específica.

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada (<ul> e <li>)

Usada para itens sem uma ordem específica.

  • Item de lista 1
  • Item de lista 2
  • Item de lista 3

Outras Tags Importantes

Outras tags essenciais para a estrutura e formatação da página.

A tag <a> cria um link. Exemplo: Acesse a DIO.

A tag <blockquote> é usada para citar um trecho longo de outro site ou autor:

A web é um sistema de informação global, interligado por hiperligações e organizado em páginas e sites.

A tag <abbr> define uma HTML abreviação.

A tag <hr> cria uma

linha horizontal.

A tag <font> modifica a aparência do texto, embora seu uso seja considerado obsoleto e o CSS seja o método preferido.
